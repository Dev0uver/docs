# Установка на Android

1. Откройте Google Play.
2. Перейдите на страницу приложения.
3. Загрузите приложение.

### Системные требования
| Параметр         | Значение       |
|------------------|----------------|
| Android версия   | 7.0+           |
| Свободная память | 50 МБ          |

Вернитесь к [основному руководству](../usage.md).
